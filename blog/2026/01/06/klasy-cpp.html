Klasy C++

Klasa w C++ wygląda następująco:

class Utripix {


};

I to jest właśnie klasa w C++. Czyli mamy tak, tzw. słowo kluczowe "class". C++ posiada słowa kluczowe które są zarezerwowane tylko na potrzeby języka C++. Kod C++ zapisujemy w plikach z rozszerzeniem .cpp i wtedy dopiero możemy je skompilować i uruchomić. W takim szablonie można powiedzieć tak, mamy słowo kluczowe "class", następnie podajemy nazwę klasy, którą sami wymyślamy, otwieramy ciało klasy nawiasem klamrowym, zamykamy i na końcu podajemy średnik. I to jest właśnie klasa C++.

class Utripix {


   public:

   int get() {
     return value;
   }

   void set(int value) {
     this->value = value;
   }


   private:

   int value = 0;


};

Czyli mamy tutaj wiele ciekawych rzeczy. Klasa, ok, a w niej public, private. Tam gdzie jest public to pod tym są metody, a pod private zmienna. Są to elementy klasy. Zmienna to pewna wartość, a metody dają do niej dostęp. Zmienna ma tak, typ, w tym przypadku int oznaczający liczby całkowite, nazwę i wartość. Tu zmienna od razu ma wartość 0, a nazwę ma "value". Nazwy można dawać dowolne, tak żeby się nie powtarzały w ramach jednej klasy czy tam zakresu. Ale typy zmiennych czyli np. int są już wbudowane i ich się trzymamy. Więc metoda zaczyna się np. od void czy int. Oznacza to typ, który zwraca. Następnie jest nazwa, argumenty i ciało metody. "return" oznacza, że metoda zwraca wynik i typ zwracany jest tutaj pokazany na samym początku metody. Jak jest void to nie ma return, bo metoda nie zwraca nic, int no to zwracamy liczbę całkowitą.

Jest to tylko przykład klasy, a nie cały program w C++. Cały program wygląda nieco inaczej, może też posiadać klasy, ale to już zupełnie inna historia.
